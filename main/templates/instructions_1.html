Phase 2 Instructions

You will be applying labels to 20 tweets about a rumor from the recent "Sydney Siege" hostage situation. You will be applying two sets of codes that define different properties about this event. You will now apply combinations of 3 different labels to each of the tweets.

Navigating Tweets

Press the up ↑ and down ↓ buttons to navigate to the previous and next tweet respectively.

Applying Labels

To apply / remove labels, you can use the keyboard keys 4 through {% if condition == 0 or condition == 1 %}6{% else %}3{% endif %}. Press the button once to add the code and a second time if you want remove it.

The second set of codes describes various properties about the tweet. These may or may not be present in the tweet, and can occur in any combination. In other words, the tweet may not exhibit any of these properties, others may exhibit some combination, and some may exhibit all of them.

  • Uncertainty
  • Ambiguous
  • Implicit
For this set, the codes can be applied in any combination. Some tweets will use none of them, some may use one or two of them, and some may use all of them.
