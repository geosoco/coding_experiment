Instructions

You will be applying labels to 45 tweets about a rumor from the "Sydney Siege" hostage situation.You will apply labels that categorize whether the tweet affirms or denies the rumor.

Suicide Bomber / Bomb belt, backpack, or vest

This rumor claims someone was a suicide bomber or that they had a bomb in a backpack, vest, or belt. You should label any tweet that supports or just passes along this information as Affirm label. Any tweet that denies or questions that presence of a bomb or the ability of the person to be a suicide bomber should be labeled Deny label. In rare cases, the tweet will not affirm or deny the rumor, but will provide information. You should label these Neutral

Applying Labels

To apply / remove labels, you can use the keyboard keys 1 through {% if condition == 0 or condition == 1 %}6{% else %}3{% endif %}. Press the button once to add the code and a second time if you want remove it.

You will be applying two sets of labels. These must be applied differently. The first set of codes describes whether the tweet affirms or denies the rumor. Only one of these may be applied at a time.
Code Keyboard Definition Example
Affirm 1 This tweet affirms, supports, and functions to pass along the story. @KevinZZ The hostage-taker has a bomb in his backpack. #Sydney #SydneySiege
Neutral 2 The tweet is exactly neutral (use sparingly). Be careful not to conflate with an implicit affirmation. RT @JillSmith: CNN has a picture of the hostage taker.
Deny 3 This tweet denies or questions all or part of the story. I don't see any sign of a bomb or backpack in that picture. #sydneysiege
