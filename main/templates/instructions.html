Instructions

You will be applying labels to 45 tweets about a rumor from the "Sydney Siege" hostage situation. In this case, rumor simply means unsubstantiated information, and it may be true or false.

The Rumor

This rumor claims someone was a suicide bomber or that they had an explosive device in backpacks, vests, or belts.

Affirmation or Denial

You will be applying {% if condition == 0 %}two sets of labels{% elif condition == 1 %}one set of labels{% else %} one label{% endif %}. These must be applied differently. The first set of labels describes whether the tweet affirms or denies the rumor. You should label any tweet that supports or just passes along this information as Affirm label. Any tweet that denies or questions the presence of a bomb or the ability of the person to be a suicide bomber should be labeled Deny label. In rare cases, the tweet will not affirm or deny the rumor. You should label these Neutral. You must apply ONLY one of these for each tweet.

Label Keyboard Definition Example
Affirm 1 This tweet affirms, supports, and functions to pass along the story. @KevinZZ The hostage-taker has a bomb in his backpack. #Sydney #SydneySiege
Neutral 2 The tweet is exactly neutral (use sparingly). Be careful not to conflate with an implicit affirmation. RT @JillSmith: CNN has a picture of the hostage taker.
Deny 3 This tweet denies or questions all or part of the story. I don't see any sign of a bomb or backpack in that picture. #sydneysiege
