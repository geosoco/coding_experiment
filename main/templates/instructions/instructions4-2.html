Navigating Tweets

Keyboard

You can navigate between tweets using the keyboard arrow keys

↑ - Previous Tweet

↓ - Next Tweet

Mouse

Two buttons can be found to the right of the tweet to navigate to the previous and next tweet.

- Previous Tweet

- Next Tweet

Applying Labels

Keyboard

Adding a label: Press keys 1 through 3 to add a label to the selected tweet.

Remove a label: Press keys 1 through 3 a second time to remove the code from the selected tweet.

Mouse

You can apply codes by clicking the label at the top left of the screen. To remove it from the currently selected tweet, click the label a second time. You can also remove it by clicking the X button on the label.

Label Keyboard Definition Example
Affirm 1 This tweet affirms, supports, and functions to pass along the story. @KevinZZ The hostage-taker has a bomb in his backpack. #Sydney #SydneySiege
Neutral 2 The tweet is exactly neutral (use sparingly). Be careful not to conflate with an implicit affirmation. RT @JillSmith: CNN has a picture of the hostage taker.
Deny 3 This tweet denies or questions all or part of the story. I don't see any sign of a bomb or backpack in that picture. #sydneysiege
