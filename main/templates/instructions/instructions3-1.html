Rumor and Labels

Rumor definition

Suicide Bomber / Bomb belt, backpack, or vest

This rumor claims someone was a suicide bomber or that they had an explosive device in backpacks, vests, or belts.

Applying Labels

You will be applying two sets of labels. These must be applied differently. The first set consists of 3 labels and only one should be applied to each tweet. The second set consists of just 1 label, and should be applied whenever you feel it applies.

Affirmation or Denial

The first set of labels describes whether the tweet affirms or denies the rumor. You should label any tweet that supports or just passes along this information as Affirm label. Any tweet that denies or questions the presence of a bomb or the ability of the person to be a suicide bomber should be labeled Deny label. In rare cases, the tweet will neither affirm nor deny the rumor. You should label these Neutral.

You must apply ONLY one of these for each tweet.
Label Keyboard Definition Example
Affirm 1 This tweet affirms, supports, and functions to pass along the story. @KevinZZ The hostage-taker has a bomb in his backpack. #Sydney #SydneySiege
Neutral 2 The tweet is exactly neutral (use sparingly). Be careful not to conflate with an implicit affirmation. RT @JillSmith: CNN has a picture of the hostage taker.
Deny 3 This tweet denies or questions all or part of the story. I don't see any sign of a bomb or backpack in that picture. #sydneysiege

Uncertainty

Additionally, another label should be applied to every tweet that expresses uncertainty about the information.

Label Keyboard Definition Example
Uncertainty 4 This tweet expresses some level of uncertainty about the story. This refers to a property of the tweet itself, not your uncertainty. In other words, the tweet suggests that the veracity of the story is not 100% resolved.
