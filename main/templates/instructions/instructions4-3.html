Instructions

Navigating Tweets

You can use either the keyboard or mouse to navigate through this task. The currently selected tweet will have a white background and a thick yellow/orange border. Applying or removing codes will only occur on this tweet.

Keyboard

You can navigate between tweets using the keyboard arrow keys

↑ - Previous Tweet

↓ - Next Tweet

Mouse

Two buttons can be found to the right of the tweet to navigate to the previous and next tweet.

- Previous Tweet

- Next Tweet

Applying Labels

Keyboard

Adding a label: Press 4 to add a label to the selected tweet.

Remove a label: Press 4 a second time to remove the code from the selected tweet.

Mouse

You can apply codes by clicking the label at the top left of the screen. To remove it from the currently selected tweet, click the label a second time. You can also remove it by clicking the X button on the label.
